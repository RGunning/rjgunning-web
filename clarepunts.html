Clare Punts

The Clare Punts booking website was developed to create a unified booking site for the whole of Clare College. Previously the Clare College MCR had a PhP booking site whilst the JCR used a paper booking system. Design requirements were to create a mobile friendly design with a simple interface that showed current booking status.

The new Clare Punts booker was created in AngularJS to create a fluid interactive site. A REST backend was created in PHP using the jacwright Restserver library. The site was completed in June 2016.

The Booker was designed to have a minimalistic feel sticking to the colours of the college. To ensure fair use of the punts, the different groups in the college body where given quotas on how many punts they could book concurrently. Limitations were also implemented on how many punts could be booked by an individual per day.

Clare College Punts home page.

The design for the homepage was to give a minimalistic view of the current status of the punts. This page is used by staff for handing out the keys, so needed to contain information on who has booked the punts, who last had the punt and who has it next. Traffic lights next to the name of the punt are used to show if the punt is available now (green), in use (yellow), or unavailable (red).

Clare College Punts make booking page.

The make booking page was designed to provide a quick select a date and show any booking conflicts for that day. A box at the top shows the term dates for the current year as boking restrictions change inside and outside of term time.

AngularJS was used to update the conflicts in real-time and to validate whether the booking is valid.

Clare College Punts my bookings page.

The my bookings page lists the upcoming bookings for a user, with a simple check box and delet button for managing bookings.

Clare College Punts Admin page for punts.

Admin page for the punts was designed to give a simple interface for adding punts, deleting punts and changing when they are available.

Clare College Punts Admin page for users.

Admin page for the users was designed to give a simple interface for adding users on bulk, deleting users and modifying users rights. A table at the bottom allows searching the database of Users.
